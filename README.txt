# Генератор Цитат

## Автор
сош №6 НЕФТЕКАМСК
## Описание
Это приложение позволяет генерировать случайные цитаты из предопределённого списка. Пользователи могут также фильтровать цитаты по автору и теме, а также сохранять и загружать историю цитат в формате JSON.

## Примеры использования
1. Сгенерировать случайную цитату.
2. Просмотреть историю сгенерированных цитат.
3. Фильтровать цитаты по автору или теме.
4. Добавить новую цитату (проверка на пустые строки).
5. Сохранить и загрузить историю цитат.
